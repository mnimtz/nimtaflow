Download & Installation

In wenigen Minuten startklar

NimtaFlow läuft per Docker auf deiner eigenen Hardware. Hol dir den Code von GitHub und starte deine Instanz.

Schnellstart (Docker)

  1. Repository holen — klone das Projekt von GitHub.
  2. .env anlegen — kopiere .env.example und setze Passwort & Foto-Pfad.
  3. Starten — docker compose up -d baut & startet alles.
  4. Quellen hinzufügen — im Web deine Foto-Ordner eintragen; Indexierung läuft automatisch.
  5. iOS-App verbinden — Server-URL eintragen, anmelden, fertig.
git clone https://github.com/mnimtz/nimtaflow.git
cd nimtaflow
cp .env.example .env   # Passwort + PHOTOS_PATH setzen
docker compose up -d

Voraussetzungen

  • Docker & Docker Compose
  • Ein Verzeichnis mit deinen Fotos/Videos
  • Optional: GPU für schnellere lokale KI
  • Optional: KI-Schlüssel (Gemini/OpenAI) für Cloud-KI
